Myntra Bag Counter

A small project that simulates a Myntra-style bag item counter using HTML, CSS, and JavaScript. This project helps in understanding DOM manipulation, event handling, and updating UI dynamically.

Features

Add items to the bag

Move items to wishlist

Add sale items

Displays the total number of items in the bag

Simple and clean user interface


Technologies Used

HTML

CSS

JavaScript


How to Run

1. Clone or download the repository


2. Open the index.html file in any browser


3. Interact with the buttons to update the bag count



Project Structure

/project-folder
│── index.html
│── style.css
└── script.js

What I Learned

How to update text content using JavaScript

How to handle click events

How to keep and update counts dynamically

Basic styling and layout with CSS
